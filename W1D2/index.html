Minestrone Soup

    ingredients description image directions

Description

Delicious vegetable soup

Minestrone Wiki

Ingredients

  • Zucchini
  • Carrots
  • Pasta
  • Tomatoes
  • Vegetable Broth
  • Parmesan Cheese
  • Beans

Directions

  1. Chop zucchini and carrots
  2. Set on stove with vegetable broth
  3. When tender add tomatoes
  4. Add salt to taste
  5. Add beans
  6. Serve with parmesan cheese
